# LIFF TEST

## Overview
LIFF(LINE Front-end Framework)を利用したテストアプリケーションとなります

## Requirement
動作確認環境
- node v18.1.0
- npm 8.8.0
- express 4.18.1

## Usage
Heroku に push して動作確認しています。
Heroku では、 Postgres を追加で指定。

LINE Developer にてLINE Login チャネルを作成してLIFFアプリケーション設定を行う必要があります
